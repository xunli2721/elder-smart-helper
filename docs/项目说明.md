# ElderSmartHelper 项目说明

## 项目概述
ElderSmartHelper 是一款专为中老年人设计的智能手机使用智能助手软件，旨在帮助中老年群体跨越数字鸿沟，轻松、安全地使用智能手机。

## 项目背景
随着智能手机功能日益复杂，界面小、图标多、操作层级深等问题让中老年用户望而生畏。学习成本高、子女远程协助不便等问题制约了中老年群体享受数字化生活的便利。

## 目标用户
- **中老年用户**（55岁以上）：智能手机使用经验有限，需要简单易用的辅助工具
- **子女/亲属**：希望远程协助父母使用智能手机

## 核心价值
1. **降低使用门槛**：简化操作流程，提供直观引导
2. **提升学习效率**：图文分步演示，手把手教学
3. **促进亲情连接**：远程协助功能，子女可实时帮助父母
4. **个性化适配**：根据用户需求调整字体大小

## 功能特色

### 1. 适老化界面设计
- 大字体、大图标、高对比度
- 简化导航，一键直达常用功能
- 无广告干扰，纯净体验

### 2. 手把手教学系统
- 覆盖打电话、微信、WiFi 等基础操作
- 图文分步演示，清晰易懂
- 分类浏览，按需学习

### 3. 远程协助功能
- 老人端截取当前屏幕发送给家人
- 家人在截图上画圈/箭头标注
- 文字消息实时沟通，指导操作步骤

### 4. 快捷功能入口
- 健康码、扫码、乘车、缴费、视频通话等常用功能
- 大图标展示，一键直达

### 5. 个性化设置
- 字体大小可调（小/中/大/超大）
- 个人信息管理

## 技术架构

### 前端（移动端）
- **框架**：Flutter（跨平台，Android）
- **语言**：Dart
- **状态管理**：Flutter 内置 setState / Provider
- **UI**：Material Design 组件，自定义适老化主题

### 后端（服务端）
- **运行时**：Node.js + Express
- **数据库**：MySQL
- **实时通信**：WebSocket（Socket.io）
- **认证**：JWT

### 数据库设计
4 张核心表：
- `users` — 用户信息（老人、家人、管理员）
- `family_relationships` — 家庭绑定关系
- `tutorials` — 教程内容（图文步骤）
- `remote_sessions` — 远程协助会话记录
