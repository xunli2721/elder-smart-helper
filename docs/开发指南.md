# ElderSmartHelper 开发指南

## 开发环境搭建

### 系统要求
- **操作系统**：Windows 10/11, macOS 10.15+, Ubuntu 20.04+
- **Flutter SDK**：3.0 或更高版本
- **Dart SDK**：随 Flutter 附带
- **Android Studio**：最新版本（用于 Android 开发和模拟器）
- **Node.js**：v16.0.0 或更高版本
- **MySQL**：8.0 或更高版本

### 环境配置步骤

#### 1. 克隆项目
```bash
git clone https://gitee.com/lzbaawso/elder-smart-helper.git
cd elder-smart-helper
```

#### 2. 后端环境配置
```bash
cd server
npm install
cp .env.example .env
# 编辑 .env 文件，配置数据库连接信息
# 初始化数据库
mysql -u root -p < ../database/schema.sql
npm run dev
```

#### 3. Flutter 环境配置
```bash
# 检查 Flutter 环境
flutter doctor

# 进入 Flutter 项目目录
cd flutter_app

# 安装依赖
flutter pub get

# 启动 Android 模拟器（或连接真机）
flutter run
```

## 编码规范

### Dart 规范
1. **命名约定**
   - 变量/函数：camelCase（如 `userName`）
   - 类名：PascalCase（如 `LoginScreen`）
   - 常量：lowerCamelCase 或 UPPER_SNAKE_CASE
   - 文件命名：snake_case（如 `login_screen.dart`）

2. **代码风格**
   - 使用 Dart 2.x 语法特性
   - 优先使用 `const` 构造函数
   - 使用箭头函数简化单表达式
   - 避免不必要的 `new` 关键字

3. **Widget 规范**
   ```dart
   class MyWidget extends StatelessWidget {
     const MyWidget({Key? key, required this.title}) : super(key: key);

     final String title;

     @override
     Widget build(BuildContext context) {
       return Container(
         padding: const EdgeInsets.all(16),
         child: Text(
           title,
           style: const TextStyle(fontSize: 18, fontWeight: FontWeight.bold),
         ),
       );
     }
   }
   ```

### 后端规范
1. **路由规范**
   ```javascript
   const express = require('express');
   const router = express.Router();
   const auth = require('../middleware/auth');

   router.post('/register', async (req, res) => { /* ... */ });
   router.post('/login', async (req, res) => { /* ... */ });
   router.get('/profile', auth, async (req, res) => { /* ... */ });

   module.exports = router;
   ```

2. **错误处理**
   ```javascript
   // 统一错误响应格式
   res.status(400).json({ success: false, message: '错误信息' });
   res.json({ success: true, data: { /* ... */ } });
   ```

## 适老化设计指南

### 视觉设计原则
1. **字体大小**
   - 正文：最小 18sp
   - 标题：24sp 以上
   - 支持动态字体大小调整（小/中/大/超大）

2. **颜色对比**
   - 文本与背景对比度 ≥ 4.5:1
   - 重要元素使用高对比色

3. **触摸目标**
   - 最小尺寸：44x44dp
   - 元素间间距：8dp 以上
   - 避免密集排列

### 交互设计原则
1. **操作简化**：一步完成一个任务，减少操作层级
2. **反馈明确**：操作后有视觉反馈，错误提示友好易懂
3. **容错设计**：支持撤销操作，提供操作确认

## 常见问题

### Flutter 环境问题
1. **Flutter doctor 报错**
   - 确保 Android Studio 已安装并配置
   - 确保 Android SDK 已安装
   - 运行 `flutter doctor --android-licenses` 接受许可

2. **Android 模拟器启动失败**
   - 在 Android Studio 的 AVD Manager 中创建虚拟设备
   - 确保 BIOS 中已开启虚拟化（VT-x/AMD-V）

3. **依赖安装失败**
   - 运行 `flutter clean && flutter pub get`
   - 检查 pubspec.yaml 格式是否正确

### 后端问题
1. **数据库连接失败**
   - 检查 MySQL 服务是否运行
   - 验证 .env 中的数据库连接配置

2. **端口占用**
   - 修改 .env 中的 PORT 配置
   - 或关闭占用 3000 端口的进程
